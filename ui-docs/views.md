# Navigation graph

Dashboard

## Views

### Dashboard view
Regions:
- Side bar (persistent)
- Main panel

Sidebar actions:
- Dashboard -> Dashboard view
- Settings -> Settings view


Main Panel:
- Days/Matches selector: Changes between days and matches for all cards below.
- List of Goals, each a goal card
- Button to create new goal.

Goal Card:
- Title: The name of the goal
- Graph: showing last hits by match or by day, according to selector. The target from the goal is displayed as a line.
    - under each match/day is a small icon (or multiple icons) for each hero of that match that the user played.


### Settings view
- Region: General

General:
- Shows Steam id.
- Button logout.

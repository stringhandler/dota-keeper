# Analysis Page: Last Hits Rows

## Description

On the analysis page, add last hits stat rows broken down by hero category:

- **All Heroes** — average last hits across all matches
- **All Carries** — average last hits for matches where the hero role is carry
- **All Cores** — average last hits for matches where the hero role is core (carry + mid)

## Acceptance Criteria

- Three new rows appear in the analysis page stats table/list
- Each row shows the average last hits per game for the relevant hero group
- Hero role classification (carry, core) should use existing role data or OpenDota role tags
- Rows should be consistent in style with existing analysis rows

# Performance Range Card: Reorder to Worst → Avg → Best

## Problem

The Performance Range card currently shows values in an order that doesn't match the natural "low to high" reading order.

## Solution

Change the display order to: **Worst → Avg → Best**

## Acceptance Criteria

- [ ] Performance Range card shows Worst first, then Avg, then Best (left to right or top to bottom depending on layout)
